# ☕ Seruput Coffee Shop - Landing Page

Landing page single-file untuk kedai kopi lokal. Dibangun dengan HTML murni, Tailwind CSS CDN, dan vanilla JavaScript.

## 🔗 Live Demo
[https://seruput-coffee.netlify.app](https://seruput-coffee.netlify.app)

## ✨ Fitur
- 🎨 Desain Mobile-First & Responsive
- 🖼️ Galeri Lightbox
- 📋 Menu Accordion Interaktif
- 📩 Form Inquiry → WhatsApp
- 🗺️ Google Maps Embed
- 🔍 SEO + Open Graph + Schema.org
- 📍 Floating CTA WhatsApp dengan animasi pulse

## 🛠️ Teknologi
- HTML5
- Tailwind CSS (CDN)
- Font Awesome 6
- Vanilla JavaScript
- Deploy di Netlify

## 📦 Instalasi
Tidak perlu build tools. Cukup clone dan buka `index.html` di browser.

```bash
git clone https://github.com/USERNAME/seruput-coffee.git
cd seruput-coffee
open index.html
